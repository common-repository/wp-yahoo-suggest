Provides suggestions while typing a search query using the Yahoo Suggest API.

== Description ==

This plugin uses the Yahoo Suggest API with an abstraction of jQueryUI's autocomplete widget to show suggestion results for Wordpress internal search queries.

It adds an AJAX call to the search form via the WordPress API and provides suggestion feedback as you type.  The user can click on the suggestion trends which results in a Wordpress search for that query.


== Installation ==

1. Download WP Yahoo Suggest.
2. Unzip the folder into the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
